# youterm v0.3 設計書 — Blur

**作成日**: 2026-04-18
**対象**: youterm v0.3
**前提**: v0.2.0 がリリース済み

## 1. 概要

ターミナルオーバーレイに `backdrop-filter: blur()` を適用し、背景の YouTube を半透明越しに見た時にガラス風のぼかし効果を与える。ぼかし強度はオプションパネルの新規スライダーで調整・永続化できる。

## 2. スコープ

### 含む

- `Settings.blur` フィールド追加(0.0 〜 1.0)
- 初期値: **0.1**(UI 表示 10%、実効 2px)
- `set-blur` reducer action
- オプションパネルに Blur スライダー(0〜100%、step 5%)
- CSS 変数 `--term-blur`(px 単位の文字列)
- `backdrop-filter: blur(var(--term-blur))` を `#terminal-root` に適用

### 含まない

- ショートカット追加(Blur 用の加減ショートカットなし。透過度用 Cmd+] / Cmd+[ はそのまま)
- プリセットテーマ、広告ブロック(v0.4 以降)

## 3. マッピング

UI 0-100%(slider value 0.0〜1.0) → CSS px 0〜20:

```ts
const blurPx = Math.round(blur * 20)  // 0 → 0, 0.1 → 2, 1.0 → 20
document.documentElement.style.setProperty('--term-blur', `${blurPx}px`)
```

## 4. 後方互換(validate ロジック変更)

**現状**: `validateAndNormalize` は任意のフィールドが欠けたり不正だと全体を INITIAL_SETTINGS にリセット。既存ユーザーが v0.3 にアップデートすると `blur` 欠落 → transparency/bgColor/lastMode まで消える。

**変更後**: フィールドごとにチェックし、不正/欠落フィールドのみデフォルトに補完、他は保持。

```ts
export function validateAndNormalize(raw: unknown): Settings {
  if (!raw || typeof raw !== 'object') return INITIAL_SETTINGS
  const r = raw as Record<string, unknown>

  const transparency =
    typeof r.transparency === 'number' && r.transparency >= 0 && r.transparency <= 0.9
      ? r.transparency
      : INITIAL_SETTINGS.transparency

  const bgColor =
    typeof r.bgColor === 'string' && VALID_COLORS.includes(r.bgColor as ColorKey)
      ? (r.bgColor as ColorKey)
      : INITIAL_SETTINGS.bgColor

  const lastMode =
    typeof r.lastMode === 'string' && VALID_MODES.includes(r.lastMode as Mode)
      ? (r.lastMode as Mode)
      : INITIAL_SETTINGS.lastMode

  const blur =
    typeof r.blur === 'number' && r.blur >= 0 && r.blur <= 1
      ? r.blur
      : INITIAL_SETTINGS.blur

  return { transparency, bgColor, lastMode, blur }
}
```

既存テスト 6 本中、`null` fallback テストは依然通るが、「フィールド不正で INITIAL_SETTINGS にフォールバック」テストが「そのフィールドだけ置換、他保持」へ変わる。テスト仕様は Task 2 で更新する。

## 6. UI(オプションパネル)

Transparency スライダーの下に Blur セクションを追加:

```html
<div class="options-section">
  <label for="opt-blur">Blur</label>
  <input type="range" id="opt-blur" min="0" max="1" step="0.05" />
  <span class="opt-value" id="opt-blur-value">10%</span>
</div>
```

値表示は **百分率**(`Math.round(v * 100) + '%'`)。透過度は小数点2桁だったが、blur は直感的な %(整数)表示にする。

## 7. CSS 変更

```css
:root {
  --term-bg-r: 0;
  --term-bg-g: 0;
  --term-bg-b: 0;
  --term-alpha: 0.75;
  --term-blur: 2px;
}

#terminal-root {
  background: rgba(var(--term-bg-r), var(--term-bg-g), var(--term-bg-b), var(--term-alpha));
  backdrop-filter: blur(var(--term-blur));
  -webkit-backdrop-filter: blur(var(--term-blur));  /* Safari prefix (Chromium ≥76 ok without prefix) */
  overflow: hidden;
  position: absolute;
  inset: 0;
}
```

`-webkit-backdrop-filter` はベンダープレフィックス(Electron 30+ / Chromium 120+ では不要だが、低リスク・低コスト保険として入れる)。
