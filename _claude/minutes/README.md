# minutes/ — 情報の一次着地点

外部から入ってくる情報の最初の着地点。
同じフォルダ内で情報が段階的に加工される: 音声ファイル → 文字起こし → 構造化議事録。

## 置くもの
- MTG録音ファイル（.m4a, .mp3等）
- 文字起こしテキスト（.txt）
- 整理済み議事録（.md）

## 命名規則
```
{YYYY-MM-DD}_{internal|external}_{会議名}.{拡張子}

例:
2026-03-13_internal_weekly.m4a        — 内部定例の録音
2026-03-13_internal_weekly.txt        — 文字起こし
2026-03-13_external_client-mtg.m4a    — 外部MTGの録音
```

## 運用ルール
- 音声ファイルは加工後も削除しない（原本保全）
- 文字起こし後のテキストも残す（スナップショット）
- Notionに議事録を登録した後も、ローカルのmdは残す
